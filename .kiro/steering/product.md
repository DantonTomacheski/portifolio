# Product Overview

**Meu Portfolio Gamificado** is a gamified portfolio website with a retro pixel-art aesthetic inspired by classic video games. The project combines nostalgic visual elements (CRT effects, pixel fonts, retro color schemes) with modern web technologies to create an engaging personal portfolio experience.

## Key Features

- Retro gaming aesthetic with pixel-art styling
- CRT monitor effect for nostalgic feel
- Custom pixel-based shadows and borders
- Gamified portfolio sections
- Responsive design with Tailwind CSS
- Custom cursor (ponteiro.png)

## Target Audience

Personal portfolio showcasing work and skills in a creative, game-inspired format.
